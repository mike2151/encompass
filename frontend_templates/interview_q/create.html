How It Works

How it works:

When a user submits their code, the interviewee's code, your support (code base) files and test files will all be moved in the same directory and the test files will be ran. You should keep this in mind when writing your code. The interviewee will be brought to the results screen with a display of the visible tests they passed. After the interviewee submits, you will be able to see the interviewee's test results and code submission.
